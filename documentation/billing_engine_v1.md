# Gabby Billing Engine V1

## Status
Working prototype

## Technology
- Next.js
- React
- Node.js

## Current Features
- SOAIP note generator
- Ledger entry generator
- Recovery-oriented language
- HIPAA-safe terminology using "client"

## Workflow
1. Enter shorthand service details
2. Generate billing note
3. Copy SOAIP note
4. Copy ledger entry

## Current Limitations
- No client database yet
- No persistent ledger storage yet
- No reporting yet
- No authentication yet
